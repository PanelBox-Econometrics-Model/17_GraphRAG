# FASE 4: Revisao e Submissao

**Duracao**: 2 meses
**Objetivo**: Revisao interna, ajustes finais, submissao ao journal
**Pre-requisitos**: FASE 3 completa (draft do manuscrito)

---

## Tarefa 4.1: Revisao Interna

**Duracao**: 2-3 semanas

### Checklist de Revisao:
- [ ] Clareza: cada secao e compreensivel por alguem de fora do projeto?
- [ ] Contribuicoes: estao claramente diferenciadas da literatura existente?
- [ ] Experimentos: sao reproduziveis? Descricao suficiente?
- [ ] Figuras: legibilidade em preto-e-branco (para impressao)
- [ ] Tabelas: formatacao consistente, legenda clara
- [ ] Referencias: completas, formatadas corretamente
- [ ] Abstrato: 250 palavras, contribuicoes claras
- [ ] Comprimento: dentro do limite do journal alvo

### Reviewers Internos:
- [ ] Autor 1: revisao completa
- [ ] Autor 2: revisao completa
- [ ] Revisor externo (se disponivel): revisao de conteudo

---

## Tarefa 4.2: Preparacao do Replication Package

**Duracao**: 1-2 semanas

### Conteudo:
- [ ] Codigo-fonte do sistema GraphRAG (anonimizado se necessario)
- [ ] Dataset de avaliacao (50+ perguntas com gold-standard)
- [ ] Scripts de reproducao para todos os experimentos
- [ ] README com instrucoes de instalacao e execucao
- [ ] Resultados brutos (JSON/CSV)
- [ ] Dockerfiles para ambiente reproducivel

### Onde hospedar:
- Opcao A: GitHub (publico ou privado, link no paper)
- Opcao B: Zenodo (DOI permanente)
- Opcao C: Ambos

---

## Tarefa 4.3: Formatacao Final

**Duracao**: 1 semana

### Por journal:
- [ ] **ACM TOIS**: ACM template (acmart), double-column
- [ ] **Information Fusion**: Elsevier template, single-column
- [ ] **EMNLP**: ACL template, double-column, 8 pages + refs
- [ ] **KDD**: ACM template, double-column, 9 pages

### Checklist:
- [ ] Template correto aplicado
- [ ] Figuras em alta resolucao (300+ DPI)
- [ ] Supplementary materials separados
- [ ] Cover letter escrita
- [ ] Checklist do journal preenchida

---

## Tarefa 4.4: Submissao

**Duracao**: 1 semana

### Passos:
- [ ] Cadastro no sistema de submissao do journal
- [ ] Upload do manuscrito
- [ ] Upload de supplementary materials
- [ ] Upload de cover letter
- [ ] Sugestao de reviewers (3-5 nomes)
  - Especialistas em GraphRAG
  - Especialistas em Knowledge Graphs
  - Especialistas em Information Retrieval
  - Especialistas em Econometria Computacional
- [ ] Confirmacao da submissao

---

## Tarefa 4.5: Pos-Submissao

**Duracao**: Variavel (3-6 meses para review)

### Enquanto aguarda:
- [ ] Publicar pre-print no arXiv
- [ ] Divulgar no Twitter/LinkedIn
- [ ] Apresentar em seminarios internos
- [ ] Preparar respostas para perguntas comuns de reviewers

### Se revisoes forem solicitadas:
- [ ] Responder ponto a ponto
- [ ] Executar experimentos adicionais se necessario
- [ ] Resubmeter dentro do prazo
